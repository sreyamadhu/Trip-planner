The Trip Planner website is a CS project aimed at providing users with a convenient platform to plan their trips. The website offers a user-friendly interface where users can input their available time and budget. Additionally users have option to specify their preferred destination. Based on these inputs, the website generates a list of tour packages that match the user's criteria. Users can further refine their search by selecting preferred landscapes such as beaches, hills, rivers, waterfalls, etc. The website also provides additional features such as nearby cafe spots near the location, information about the location, itinerary suggestions, and information on restaurants and accommodations. Furthermore, users can contribute to the website by adding new tourist spots to the user-driven database. Along with this user can review their travel experience. The Trip Planner website simplifies the trip planning process, offering users a centralized platform to access all the necessary information for their travel plans.
